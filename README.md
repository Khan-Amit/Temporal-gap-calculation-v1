## What This Is

This is not a calculator. It is a **temporal decision engine** based on a novel ternary model where:

- **Tp (past)** = known territory, memory, learned information (intensity 1–9)
- **Tf (future)** = unknown territory, potential, to be created (intensity 1–9)
- **H (helix width)** = the breadth of parallel possibilities (1–9)
- **The gap** = the present moment, where Tp and Tf meet, producing a **probability** of outcome 1.
- **Outcome** = a random sample (0 or 1) – the realised reality.
